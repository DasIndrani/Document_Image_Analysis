# Document_Image_Analysis

# Description:
•	The “Document Image Analysis” repository houses a system designed to extract information from document. In this project we have used generative ai for extract information and Streamlit for build web-app. This system enable to extract desired information for improved efficiency and decision-making in document-intensive workflows.

## API Key:
Create a Google API key to call Gemini Vision Pro model.


## Install
### Requirements:
•	Python 3.11
•	streamlit
•	googlegenerative-ai
•	boto3
•	python-dotenv
•   dump



## Setup:
### Docker setup in EC2:
•	sudo apt-get update -y
•	sudo apt-get upgrade
•	sudo apt-get install awscli
•	curl -fsSL https://get.docker.com -o get-docker.sh
•	sudo sh get-docker.sh
•	sudo usermod -aG docker ubuntu
•	newgrp docker
### Run Self-Hosted runner :
•	In GitHub action you need to run Self-hosted runner in ubuntu(Linux).
 

## Run in local system: 
•   streamlit run app.py
